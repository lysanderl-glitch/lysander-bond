# D-2026-0514-001: Harness English Translation & Response Language Policy

## Decision
Translate CLAUDE.md and top 10 highest-impact harness files from Chinese to English to reduce per-session token consumption by ~75%. Add Response Language Policy as a P0 constraint.

## Scope
- CLAUDE.md (1 file)
- .claude/harness/ files (10 files): product-routing, claude-md-edit-checklist, pmc-governance, ceo-guard-tests, knowledge-push-protocol, context-injection-sop, execution-env-rules, knowledge-retrieval-protocol, synapse-effectuation-governance, km-interface-health-checklist

## Rationale
- Chinese characters consume ~5x more tokens than equivalent English content
- CLAUDE.md alone saves ~2,700 tokens per session (guaranteed, always loaded)
- Full Option B scope saves ~11,768 tokens per session
- Previous attempt (Synapse-Mini-EN directory) was lost when deprecated in D-2026-0503-001
- Response Language Policy added to ensure user continues to receive Chinese output despite English file content

## Response Language Policy (P0, new)
Always respond to the President in Chinese (Simplified Mandarin).
Internal reasoning, agent prompts, file contents: English.
Quoted behavioral strings (greetings, identity phrases): preserved as Chinese literals in English files.

## QA Result
98.3/100 — Pass (threshold ≥90/100)

## Maintenance Rule Added
New harness files default to English. Chinese quoted behavioral strings are the only permitted exception.
