3. **记忆能力从 Agent 侧向基础设施侧下沉**：Anthropic SDK agent-memory beta + Supabase OpenCode 集成均指向同一趋势——上下文管理与记忆能力正从应用层向基础设施层迁移。Synapse OBS 层设计需重新审视与官方记忆 API 的边界划分，避免重复造轮子。
