We Chose Reliability Over Flexibility — This Was the Right Call

Dimension LangGraph Synapse FSM (chosen)
Execution Path Dynamic graph, non-deterministic 8 fixed states, fully deterministic
Testability Path-dependent, complex to validate Each state independently testable
Enterprise Fit Research-first design Production-first design
Audit Support Requires additional build Native, built-in, 30+ event types
Predictability Low — runtime behavior varies High — same input → same path
"ADR-0001: We reject LangGraph. In enterprise production, determinism is non-negotiable."
This decision is a trust signal — surfaces specific architecture choices (RBAC / pgvector / audit log) rather than vague maturity claims.
