## Page 1 — 产品概览：让 AI 成为可信赖的工作伙伴，而非黑盒工具

### 核心价值主张

Synapse Platform 解决的是 AI 落地中最棘手的信任问题：**AI 生成的结果，业务团队不敢直接用。**

原因不是 AI 不够智能——而是 AI 的执行过程不可见、不可控、不可审计。

**Synapse Platform 的设计哲学：AI 全力生成，人类必须审批。这不是流程规定，是技术硬约束。**

### 三大核心价值

> **[价值 1] 零工具迁移成本**
> 完全构建在 Slack 之上。用户不需要学习新工具、切换新界面——在现有 Slack 频道内完成全部 AI 协作。

> **[价值 2] AI 生成 + 人类审批，强制双保险**
> 所有 AI 输出必须经过人类审批门（Mandatory Human Approval Gate）才能生效。这是技术层面的强制约束，不可绕过。

> **[价值 3] 内联编辑即训练数据，AI 持续自我改进**
> 在 Slack 中对 AI 输出进行的每一次修改，都自动成为 4 层个人训练系统的输入，AI 会越来越懂你的风格和偏好。

## Page 2 — 20 项核心能力 & 技术架构

### 四层能力矩阵

**[层 1] 意图理解（3 项能力）**

| 能力 | 说明 |
|------|------|
| 自然语言意图解析 | 将老板/PM 的业务语言转化为结构化执行指令 |
| 双语无缝处理（EN↔ZH） | 中英文输入统一处理，多 Agent 协作无语言壁垒 |
| 上下文记忆与关联 | pgvector + Voyage AI 语义向量存储，跨会话记忆关联 |

**[层 2] 执行与协作（7 项能力）**

| 能力 | 说明 |
|------|------|
| FSM 有限状态机引擎（自研） | 确定性、可审计的状态流转，非 LangGraph，不依赖外部框架 |
| Slack Socket Mode 实时交互 | 无公开端口，安全的实时双向通信 |
| 多 Agent 协调编排 | 多个专业 AI 角色协同处理复杂任务 |
| 异步任务队列 | 长时任务不阻塞交互，后台执行完成后推送结果 |
| 高质量交付物自动生成 | 文档 / 分析报告 / 内容草稿 / 执行计划等 |
| 内联编辑捕获机制 | Slack 内修改自动记录，无需额外操作 |
| Jina AI 内容增强 | 搜索增强与内容质量提升 |

**[层 3] 治理与质量（5 项能力）**

| 能力 | 说明 |
|------|------|
| 强制人类审批门（技术约束） | 所有 AI 输出必须通过，不可绕过，不可关闭 |
| 完整执行日志与审计追踪 | 每个动作有记录，可回溯，可审计 |
| RBAC 权限管理（Phase 5） | 角色级访问控制，即将上线 |
| 错误检测与自动恢复 | 失败场景自动处理，不依赖人工干预 |
| 质量评分机制 | AI 输出质量数值化评估 |

**[层 4] 智能进化（5 项能力）**

| 能力 | 说明 |
|------|------|
| 4 层个人训练系统 | 风格偏好 · 领域知识 · 反馈模式 · 输出规范，全面对齐 |
| 自动训练样本收集 | 每次 Slack 内联编辑即样本，零额外操作成本 |
| 语义向量知识库 | pgvector 存储，Voyage AI 嵌入，语义检索而非关键词匹配 |
| 跨会话上下文持久化 | 会话结束不丢失知识，随时恢复 |
| 持续自我优化 | 随使用积累，AI 输出与用户期望差距持续收窄 |

### 技术架构一览

| 组件 | 技术选型 | 设计原因 |
|------|----------|----------|
| 接入层 | Slack Socket Mode | 无公开端口，安全实时，零用户学习成本 |
| 状态引擎 | 自研 FSM（非 LangGraph） | 确定性行为，完整可审计，不依赖外部框架 |
| AI 核心 | Claude claude-sonnet-4-6 | 高质量生成，强指令遵循 |
| 向量存储 | pgvector | 语义记忆，无缝扩展 |
| 嵌入模型 | Voyage AI | 高质量语义表征 |
| 内容增强 | Jina AI | 搜索质量与内容丰富度 |
| 审批约束 | 技术硬约束层 | 非流程配置，代码级强制，不可绕过 |

### 设计原则声明

> **[PRINCIPLE]** Synapse Platform 的核心设计原则：**AI 服务于人，永不替代人。**
>
> 人类审批门是技术层面的硬约束——这不是可以关闭的配置开关，而是系统架构中不可绕过的执行路径。
>
> 这个设计选择确保了 AI 能力的充分发挥与人类控制权的完整保留之间的平衡。
